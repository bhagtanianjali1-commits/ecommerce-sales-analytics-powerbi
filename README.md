# Content & Engagement Analytics Dashboard (Power BI)

This project presents an interactive Power BI dashboard built to analyze content performance, audience engagement, and release trends using IMDb-based data.

The dashboard focuses on understanding how runtime, ratings, votes, content type, age certification, and release year relate to audience engagement.
